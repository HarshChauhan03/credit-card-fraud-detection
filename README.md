# 💳 Credit Card Fraud Detection

## 📌 Overview

This project predicts whether a credit card transaction is fraudulent or genuine based on transaction details.

It demonstrates a classification-based machine learning workflow used in banking and financial security systems.

---

## 🎯 Objective

- Detect fraudulent transactions
- Learn classification algorithms
- Understand fraud analytics

---

## 🧠 Workflow

```text
Transaction Data
↓
Data Preprocessing
↓
Random Forest Classifier
↓
Fraud Detection
```
